🔹 Descrição:
Projeto desenvolvido em ASP.NET MVC para gerenciamento de clientes, com operações CRUD (Create, Read, Update, Delete). Inclui:
Views (telas) para listagem, cadastro, edição e exclusão.
Integração com Entity Framework para persistência em banco de dados.
Validações de formulário e layout responsivo.

✨ Funcionalidades:
✔️ Listagem de clientes (Index.cshtml) com tabela dinâmica.
✔️ Cadastro e edição de dados com validação.
✔️ Exclusão segura com confirmação.
✔️ Design organizado com Bootstrap.

🛠️ Tecnologias:
Back-end: ASP.NET MVC, C#, Entity Framework.
Front-end: Razor Pages, HTML5, CSS3, Bootstrap.
Banco de dados: SQL Server (ou LocalDB).

🚀 Como executar:
Clone o repositório.
Restaure os pacotes NuGet (Update-Package no PMC).
Execute as migrations (Update-Database).
Acesse http://localhost:porta/Cliente.

📌 Pré-requisitos:
Visual Studio 2022+.
.NET Framework 4.7+ ou .NET Core 6+.
